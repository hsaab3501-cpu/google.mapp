# ملخص مشروع المتتبع الخفي (Stealth Tracker) 🚀
**تاريخ النسخة:** 26 فبراير 2026
**الحالة:** يعمل بكفاءة عالية (الإصدار النهائي المستقر).

## 🛠️ تفاصيل المشروع والميزات الحالية:
هذا المشروع هو عبارة عن متتبع GPS متقدم ومخفي على هيئة رابط مشاركة "خرائط جوجل". المميزات المتوفرة في هذه النسخة:
1. **الواجهة (Frontend - index.html):**
   - تصميم مطابق 100% لعرض موقع من Google Maps.
   - نظام كسر المتصفحات الداخلية (Instagram / Facebook) بإجبار الأندرويد على فتح متصفح Chrome.
   - إذا كان الـ GPS مغلقاً في هواتف الأندرويد، تظهر شاشة سوداء وتوجه الضحية لفتح الإعدادات تلقائياً (عبر Intent) بعد ثانيتين.
   - إرسال فوري ومزدوج للبيانات لضمان وصول التقرير (بدون بطء أو تأخير في التحويل).
   - جلب وتخزين بيانات دقيقة: (البطارية، الرام، المعالج، الشاشة، نظام التشغيل، التوقيت المحلي).

2. **السيرفر والبوت (Backend - server.js):**
   - يعمل بتقنية Express.js و HTTPS.
   - نظام الرد السريع: السيرفر يرد على المتصفح بـ 200 OK فوراً ليتحول المستخدم لخرائط جوجل بلا تأخير، بينما يستمر السيرفر في الخلفية بجلب بيانات الـ IP وإرسالها لتليجرام.
   - تقرير تليجرام شامل ومفصل (باللغة العربية) يتضمن إحداثيات GPS الدقيقة، بيانات الـ IP، ومواصفات الجهاز كاملة.

## ⚙️ متطلبات التشغيل على الجهاز الجديد:
لكي يعمل المشروع على جهاز آخر أو يتم رفعه (Deploy) على منصة مثل Render، يجب التأكد من:
1. تثبيت **Node.js**.
2. تشغيل أمر `npm install` لتحميل المكتبات (`express`, `dotenv`).
3. التأكد من وجود ملف `.env` يحتوي على:
   TELEGRAM_BOT_TOKEN=التوكن_الخاص_بك
   TELEGRAM_CHAT_ID=آيدي_حسابك
4. لتشغيل السيرفر محلياً للمعاينة استخدم أمر: `node server.js`

## 💡 الخطوات القادمة والملاحظات:
- المشروع جاهز تماماً للرفع (Push) على منصة GitHub وإعادة ربطه بـ Render.
- خوارزمية الاتصال بالتليجرام مبرمجة لتعمل كـ "خلفية" (Background process) لمنع المستخدم من الإحساس بأي تأخير عند الضغط على زر "السماح".
- الرابط النهائي للتحويل محفوظ كمتغير `finalUrl` في ملف `index.html` ويمكنك تغييره متى شئت.
